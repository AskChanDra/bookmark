# SASS

##### Install

- NodeJS : npm

```bash
npm install -g sass
```

- Windows

```bash
choco install sass
```

- Mac OS

```bash
brew install sass/sass/sass
```

##### SCSS to CSS

```bash
sass source/stylesheets/index.scss build/stylesheets/index.css
```
